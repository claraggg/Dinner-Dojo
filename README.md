# Dinner-Dojo

## Description

Dinner Dojo is a program that recommends recipes based on ingredients a user already has. The user first selects a meal category, then enters a list of ingredients. The program uses TheMealDB API to fetch recipes from the selected category and compares each recipe’s ingredients with the user’s ingredients.

The program then outputs:
- the best recipe suggestion (based on ingredient match)
- the ingredients missing from that recipe
- recipes the user can make exactly
- recipes the user is close to making

---

## How to Run

1. Make sure you have Python installed
3. Run the program: python3 dinner_dojo.py
4. Follow the prompts
